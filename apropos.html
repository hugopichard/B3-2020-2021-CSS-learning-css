Bonjour, je m'appelle Hugo Pichard, je suis en 3éme année de Bachelor Chef de Projet digital à l'ECV Digital Bordeaux.

Le projet.

Ce projet a été réalisé dans le cadre de ma formation. Il vise à nous sensibiliser aux bases du CSS tout en respectant la sémantique et les nouvelles techniques de ce langage afin de comprendre et appréhender le développement web, des compétences obligatoires dans le domaine du digital. Les langages utilisés dans ce projet sont le HTML et le CSS, basé sur la structure "Barebones V3" lui meme basé sur Skeleton de Dave Gamache. J'ai utilisé différentes techniques afin d'appréhender toutes les facettes du CSS, la mise en page est faite avec css grid (page personalités) mais aussi avec flexbox (fiches personalités). Les images sont gérées des fiches personalités sont gérées avec object fit et le site respecte la sémantique HTML5. L'intégralité du site est responsive ave des breakpoints mediaqueries.
